Place your .lottie animation files in this folder. Name the CTA animation `schedule-cta.lottie` to match the HeroSection component.

You can export a .lottie from LottieFiles or convert a Lottie JSON to .lottie using the DotLottie app. The `DotLottieCTA` uses the web component from LottieFiles CDN to render the file client-side.
